LAP PROCESS DISCOVERY - ÍNDICE DE DOCUMENTOS
=============================================

OBJETIVO
========
Conjunto completo de documentación para levantar procesos en La Patagua Mining
y diseñar implementación óptima de n3uralia ERP.

ESTRUCTURA DE ARCHIVOS
======================

Cada documento de departamento (3-6) contiene:

┌─ OBJETIVO
├─ PARTICIPANTES
├─ CONTEXTO ACTUAL
├─ 6-8 PREGUNTAS PRINCIPALES
├─ INFORMACIÓN A RECOLECTAR (checklist)
├─ MATRIZ DE PROCESOS
├─ RACI MATRIX
├─ PAIN POINTS POTENCIALES
└─ INTEGRACIÓN CON n3uralia

CÓMO USAR ESTOS DOCUMENTOS
==========================

PASO 1: PREPARACIÓN (Semana previa)
───────────────────────────────────

Facilitador:
1. Lee LAP_00_GUIA_FACILITADOR.txt (1h)
2. Lee LAP_01_DISCOVERY_GUIDE.txt (30min)
3. Lee los 3 documentos de la primera semana (1.5h)
4. Imprime matrices y preguntas
5. Envía agendas a participantes con 48h anticipación

PASO 2: REUNIÓN 1 - OPERACIONES (2h)
─────────────────────────────────────

Facilitador abre LAP_02_OPERACIONES.txt:
1. Introduce objetivo (5min)
2. Mapea procesos usando preguntas (60min)
3. Valida matriz con participantes (40min)
4. Captura fotos de pizarra (5min)
5. Cierra (10min)

Resultado: Matriz completada + Pain points validados

PASO 3: DOCUMENTACIÓN (24h post-reunión)
─────────────────────────────────────────

Facilitador:
1. Limpia notas
2. Completa matriz
3. Documenta pain points
4. Envía resumen a participantes para validación
5. Obtiene firma de validación

PASO 4: SÍNTESIS (Después de 6 reuniones)
──────────────────────────────────────────

Facilitador:
1. Lee todos los pain points capturados
2. Llena LAP_07_MATRIZ_PROCESOS_CONSOLIDADA.txt
3. Identifica 12-15 procesos CRÍTICOS
4. Mapea dependencias
5. Propone fases de implementación
6. Valida con La Patagua

PASO 5: PRESENTACIÓN
─────────────────────

Usa estructura de LAP_00_GUIA_FACILITADOR.txt:
1. Resumen ejecutivo (10min)
2. Estado actual (15min)
3. Procesos críticos (15min)
4. Roadmap (15min)
5. Próximos pasos (5min)

PREGUNTAS CLAVE POR DEPARTAMENTO
=================================

OPERACIONES:
- ¿Cuál es el ciclo diario completo?
- ¿Cuánto toma el ciclo? ¿Debería tomar menos?
- ¿Qué información se pierde?

MANTENIMIENTO:
- ¿Cómo reciben solicitudes de Operaciones?
- ¿Cuánto toma de incidente a resolución?
- ¿Cómo priorizan OT?

BODEGA:
- ¿Qué tiempo toma despacho desde solicitud?
- ¿Se cumple FIFO siempre?
- ¿Hay diferencias entre sistema y físico?

DOCUMENTACIÓN:
- ¿Cuántos documentos vencen cada mes?
- ¿Alguien falta renovación?
- ¿Cómo se comunican cambios de procedimientos?

FINANZAS:
- ¿Cuánto tiempo toma cierre de mes?
- ¿De dónde vienen los costos?
- ¿Presupuesto vs real: qué varía?

RRHH:
- ¿Cuánto toma procesar nómina?
- ¿Asistencia biométrico o manual?
- ¿Certificaciones vencidas se controlan?
