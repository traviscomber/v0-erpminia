## 1. RESUMEN EJECUTIVO

Se propone el desarrollo de **n3uralia ERP**, una plataforma enterprise de gestión operacional específicamente diseñada para empresas mineras y contratistas en Chile.

**Valor de Propuesta:**
- Integración de 5 módulos operacionales en una única plataforma
- Cumplimiento automático con regulaciones SERNAGEOMIN
- Reducción de costos operacionales del 15-25%
- Implementación en 7 meses
- Soporte y mantenimiento incluido por 1 año

---

## 2. ALCANCE DEL PROYECTO

### Módulos Incluidos:

1. **Módulo Producción** - Telemetría en vivo (50+ sensores, alertas automáticas)
2. **Módulo Mantención** - Gestión de órdenes de trabajo, MTTR, históricos
3. **Módulo Bodega** - Inventario con FIFO, QR scanning, reorden automático
4. **Módulo HSE** - Compliance SERNAGEOMIN, reportes de incidentes, auditorías
5. **Módulo Documentos** - Gestión centralizada, versionado, workflows de aprobación

### Características Técnicas:

- **Frontend:** Next.js 16 + React 19 + TypeScript
- **Backend:** API REST + Server Actions
- **Database:** Supabase (PostgreSQL) con RLS
- **Autenticación:** Supabase Auth + JWT
- **UI Components:** shadcn/ui (60+ componentes)
- **Hosting:** Vercel (CDN global, Edge Functions)
- **Monitoreo:** Sentry + Vercel Analytics

### No Incluido:

- Integración con SAP/Oracle (disponible en fase 2)
- Apps móviles nativas (web responsive incluida)
- Entrenamiento presencial (virtual incluido)
- Hardware o licencias de terceros
