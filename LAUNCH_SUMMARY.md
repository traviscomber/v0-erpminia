## Technical Architecture

**Stack Choices:**
- Frontend: Next.js 16 + React 19 (Latest stable)
- Styling: Tailwind CSS v4 + shadcn/ui
- Database: Supabase PostgreSQL (100% managed)
- Charts: Recharts (optimized performance)
- Forms: React Hook Form (lightweight)
- Language: TypeScript strict mode
- Deployment: Ready for Vercel

**Design Decisions:**
- Mobile-first responsive design
- Dark theme for mining operations (eye strain reduction)
- Modular component architecture
- Service layer for business logic separation
- REST API for third-party integrations
- Audit logging for compliance

## Deployment Instructions

1. **Setup Database:**
   - Visit /setup/initialize-db
   - Enter ADMIN_INIT_TOKEN
   - Click Initialize

2. **Configure Environment:**
   - Add Supabase credentials
   - Set production domain
   - Enable SSL

3. **Deploy to Vercel:**
   - Connect repository
   - Set env variables
   - Auto-deploy on commit

4. **Monitor & Verify:**
   - Check uptime
   - Verify all endpoints
   - Test user workflows
